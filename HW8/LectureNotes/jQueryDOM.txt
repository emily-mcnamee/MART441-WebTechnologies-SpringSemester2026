What about the DOM? Remember document.getElementById()? How do we access DOM elements in jQuery?

To access or change items in the DOM, there are three main methods you’ll use:

text()
html()
val()

These allow us to get the text from an element, get the text including any HTML markup, or get the value from a textbox (for instance).


Example: text()
---

Let’s see these in action.

<!DOCTYPE html>
<html>

<head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
    <script>
        $(document).ready(function () {
            $("button").click(function () {
                window.alert($("#infoid").text());
            });
        });
    </script>
</head>

<body>

    <p id="infoid">This is a <b>paragraph</b>.</p>
    <p class="myGreatClass" style="background-color:red">This is the second paragraph.</p>
    <button>Click me to get information</button>
    <br>
    <div id="message"></div>

</body>
</html>

This code should return just the inner text of the paragraph tag.

Example: val()
---

Finally, if we want to get something out of a text field, we can use the val() method.

<!DOCTYPE html>
<html>

<head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
    <script>
        $(document).ready(function () {
            $("button").click(function () {
                window.alert($("#txtName").val());
            });
        });
    </script>
</head>

<body>

    <p id="infoid">This is a <b>paragraph</b>.</p>
    <p class="myGreatClass" style="background-color:red">This is the second paragraph.</p>
    <input type="text" id="txtName">
    <button>Click me to get information</button>
    <br>
    <div id="message"></div>

</body>
</html>


Changing Content with text(), html(), and val()
---

What if you want to change the text of a tag in the DOM? Before, we used the equals sign and updated innerHTML. With jQuery, you can change content by passing a parameter into the method.

For example:

<!DOCTYPE html>
<html>

<head>
    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
    <script>
        $(document).ready(function () {
            $("button").click(function () {
                $("#infoid").text($("txtName").val());
            });
        });
    </script>
</head>

<body>

    <p id="infoid">This is a <b>paragraph</b>.</p>
    <p class="myGreatClass" style="background-color:red">This is the second paragraph.</p>
    <button>Click me to get information</button>
    <br>
    <div id="message"></div>

</body>
</html>

You can do that with the html() and val() methods as well.
